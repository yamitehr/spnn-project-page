Abstract

The Moore-Penrose Pseudo-inverse (PInv) is the fundamental solution for linear systems. We generalize PInv to the non-linear regime and introduce Surjective Pseudo-invertible Neural Networks (SPNN), a class of architectures designed to admit a tractable non-linear pseudo-inverse. Our construction satisfies reflexive Penrose-style consistency and supports null-space projection. Building on this, we formalize Non-Linear Back-Projection (NLBP), a method that enforces consistency constraints for non-linear surjective mappings. We leverage SPNNs and NLBP to expand zero-shot inverse problems: extending diffusion-based null-space projection from linear degradations to broad non-linear degradations (from optical distortions to semantic abstractions like classification), enabling zero-shot inversion and precise semantic control without retraining the diffusion prior.

We define NLBP as a closed-form projection in the completed space that guides the diffusion process at each timestep by enforcing semantic consistency while preserving null-space structure. Given an estimate $x$ and a target measurement $y$, the update is $$ x' = G^{-1}\!\left( G(x) - G\!\big(g^\dagger(g(x))\big) + G\!\big(g^\dagger(y)\big) \right), $$ which guarantees $g(x') = y$ while leaving the null-space component invariant in the induced $G$-metric.

Key Results

Reconstruction from Semantics

Using only a low-dimensional semantic measurement (e.g., 40 attribute logits), NLBP-guided diffusion reconstructs photorealistic faces that match the target semantics without retraining.

Attribute-Controlled Generation

We can enforce a single attribute (e.g., ‚ÄúEyeglasses‚Äù) by modifying only one coordinate in the semantic space and applying gentle NLBP guidance, while leaving all other attributes and textures free for the diffusion prior to hallucinate.

Multi-Attribute Conditional Generation

We simultaneously enforce multiple semantic constraints (e.g., Male + Smiling + Eyeglasses) by fixing several coordinates in the target logit vector and applying NLBP guidance during sampling. This effectively guiding the diffusion process to the intersection of these semantic manifolds.
